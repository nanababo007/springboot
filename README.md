# springboot
sample rest server api in springboot framework

my react, reactjs, angular, angularjs, vue, vuejs projects will use this rest server api common project.
each client projects use there own folder.

# 개발환경
- 스프링부트 : 버전 3.1.0
- ui : thymeleaf
- 로그인 : 스프링 시큐리티
- 디비 서버 : mysql
- 디비 처리 : mybatis, jpa
- 소스 헬퍼 : lombok
